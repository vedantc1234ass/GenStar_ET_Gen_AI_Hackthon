# GenStar – Enterprise AI Workflow System

  Built for the ET Gen AI Hackathon.

  A fully functional multi-agent AI workflow system with:
  - Meeting intelligence (transcript → AI task extraction)
  - Customizable workflow manager
  - Real-time analytics dashboard
  - Employee performance tracking
  - AI Chatbot (enterprise-grade)
  - 8-agent pipeline: Orchestrator → Decision → Data → Action → Verification → Recovery → Audit → Monitoring

  ## Stack
  - **Frontend**: React + Vite + Tailwind CSS + Recharts + Framer Motion
  - **Backend**: Node.js + Express 5
  - **Database**: PostgreSQL + Drizzle ORM
  - **API**: OpenAPI 3.1 with code generation
